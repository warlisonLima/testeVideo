Exemplos de elementos para formatar textos

Marcações básicas

strong

O elemento <strong>; é usado para uma parte do texto. Ele funciona de maneira similar ao elemento <b>, fazendo o texto ficar em negrito, mas ele tem valor semântico.

em

O elemento é também é usado para enfatizar uma parte do texto. Visualmente ele produz um resultado similar ao elemento <i>, colocando um trecho em italico, mas ele tem valor semântico.

u

O efeito de sublinhado pode ser obtido usando o elemento <u>

Outras marcações

mark

O elemento é usado para destacar partes do texto Não esqueça de comprar milho hoje!

small

O elemento <small> (um texto menor), mas tem função semântica

Elementos del e ins

O <del> indica que o texto foi ou deve ser deletado de um determinado conteúdo. O <ins> é responsável por marcar algum texto que foi inserido em um determinado conteúdo desde sua versão anterior Minha cor favorita é azul amarelo!

Elementos sub e sup

<sub> é usado para textos subescrtios, aqueles que devem aparecer ligeiramente menor e abaixo que o restante do texto. Já o <sup> marca textos sobrescritos, isto é aqueles que devem aparecer ligeiramente acima dos demais.
