Criando tabelas no HTML

A tabela é uma ferramenta utilizada para organizar dados em linhas e colunas e permitir uma melhor visualização das informações que estão sendo apresentadas.

A seguir temos uma tabela feita em HTML para conhecermos melhor os elementos básicos usados para construí-las.

Os elementos básicos: tr e td

O tr é o elemento responsável por inserir linhas em uma tabela.

Para criarmos as células, é necessário usar o td

Uma tabela mais organizada

Para conseguirmos criar uma tabela ainda melhor, é necessário acrescentar mais alguns outros elementos à nossa tabela. No caso esses elementos serão thead, th, caption e tbody

thead:
Serve para marcar o cabeçalho da tabela, que geralmente é a sua primeira linha.
th:
Usado para definir os valores dos títulos das colunas. É utilizado apenas uma vez em cada tabela.
caption:
Trata-se de um elemento com valor semântico que define a legenda da tabela.
tbody:
Ele serve para marcar as partes da tabela que não fazem parte do cabeçalho e nem do rodapé, é marcado com tfoot.
